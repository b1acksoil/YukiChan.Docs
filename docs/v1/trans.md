# BaiduTranslate - 百度翻译

## 指令帮助

### #trans - 执行百度翻译

**指令格式**

```
#trans [源语言:目标语言] <文本>
```

**简写形式**

`翻译`

**使用示例**

```
- 将 pure 翻译成中文（源语言将自动检测）
#trans pure
- 等同于
#trans auto:zh pure

- 将 rabbit 由英文翻译成日文
#trans en:jp rabbit
- 等同于
#trans 英语:日语 rabbit

- 使用简写形式
翻译 world ender
```
