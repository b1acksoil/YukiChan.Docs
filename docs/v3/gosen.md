# Gosen - 五千兆円图片生成

## 指令帮助

指令格式中，中括号 `[]` 中的内容为**可选参数**，尖括号 `<>` 中的内容为**必选参数**。

### /5k - 生成五千兆円图片

**指令格式**

```
/5k <上方文本> <下方文本>
/gosen <上方文本> <下方文本>
五千兆 <上方文本> <下方文本>
```

**选项说明**

`-o, --offset`: 指定下方文本的水平偏移量，默认为 0

::: details 使用示例
- 生成一张五千兆图片
```
五千兆 杂鱼♡ 杂鱼♡
```
